VK-GTK 0.3
======

vk-gtk is notifications for vk.com (vkontakte.ru).


INSTALL
=======

 Building vk-gtk from source requires that several dependencies are installed.
 These include:
       cmake
       g++
       libc6-dev 
       libgtk2.0-dev
       libglade2-dev
       libnotify-dev
       libboost-signals-dev
       libcurl4-gnutls-dev
       libwebkitgtk-dev or libwebkit1.0-dev

 Run the command:
       cmake ./ && make

 Installation:
       make install


RUNNING
=======

 Run the command:
       vk-gtk
